### Создатель рамок на картинку

Приложение рисует рамку вокруг картинок.

Выберите нужное изображение (изображения), укажите размер рамки в пикселях. Выбранные файлы будут отображены в списке.
Рамка будет нарисована вокруг изображения с одинаковыми полями. Пока что доступен только белый цвет рамки.

### Зачем нужна эта программа?

Чтобы нарисовать рамку к выбранной картинке раньше необходимо было снова открывать фотошоп,
запускать экшены, перерисовывать. Эта программа несколько ускоряет этот момент.
